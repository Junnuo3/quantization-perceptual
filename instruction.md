Build a simple website for perceptual comparison of quantization settings.

The project has an `audio/` folder containing 5 example folders. Each example folder includes audio files generated by quantizing only one attribute at a time: `duration`, `onset`, or `velocity`, while keeping all other musical elements unchanged.

For each of the 5 examples, create a separate comparison block on the webpage. Inside each block, include three sections:

1. **Duration quantization**
2. **Onset quantization**
3. **Velocity quantization**

Each section should display all corresponding audio files for that quantization type and provide an audio player for each file, so users can listen and compare the perceptual differences between quantization levels.

Requirements:

* Automatically read or organize files from the `audio/` directory if possible.
* Use a clean and simple UI.
* Clearly label each example, quantization type, and quantization level.
* Use standard HTML audio players.
* The website should be able to run on github.io with minimal setup.
* Keep the implementation simple and easy to modify.
